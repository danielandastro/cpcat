# cpcat

cpcat (or copycat) is a lightweight unix utility written in Rust that copies the contents of a file directly to your clipboard. 

# Usage
<b> Default: </b>
`cpcat` file.txt


<b> Verbose: </b>
`cpcat` -v file.txt

By default, `cpcat` succeeds silently.
## Install from AUR
`yay -S cpcat`
## Install with homebrew
`brew install danielandastro/tools/cpcat`
## Manual Installation

1. Compile the optimized release binary (recommended) OR download a binary from the releases page
2. Copy it to ~/.local/bin/ or anywhere on your PATH
