# ccat

ccat (or copycat) is a lightweight unix utility written in Rust that copies the contents of a file directly to your clipboard. 

# Usage
<b> Default: </b>
`ccat` file.txt


<b> Verbose: </b>
`ccat` -v file.txt

By default, `ccat` succeeds silently.

## Installation

1. Compile the optimized release binary (recommended) OR download a binary from the releases page
2. Copy it to ~/.local/bin/ or anywhere on your PATH
